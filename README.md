# Fitness Tracker (NoSQL Project)

This is a Full Stack Fitness Tracker Web Application built using NoSQL technologies.

## Technologies Used
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB Atlas (NoSQL)
- Authentication: JWT

## Features
- User Registration
- User Login
- Add Workout
- View Workouts
- Data stored in MongoDB Atlas

## Project Structure
fitness-tracker-nosql/
│
├── backend/
├── frontend/
└── README.md

## How to Run the Project

### Backend
cd backend
npm install
node src/server.js

### Frontend
cd frontend
npm install
npm start

## Architecture
React Frontend → Node.js Backend → MongoDB Atlas Database
